Non-Working group email lists

While email lists are a key tool for IETF working groups, email lists also are often used to discuss particular topics of interest to the IETF community.

    In this section Hide in this section
    • Community discussions
    • Workshops
    • Non-Working group email lists
    • Birds of a Feather

    Key Info

    These can be lists set up specifically for that topic, lists that have belonged to former working groups but are still active, lists that are specified in the registration rules of some parameter registry as "for review," and others.

    Mail sent to all these lists is considered an IETF "Contribution" as defined in RFC 5378, Section 1 (see also the NOTE WELL statement). Refer to these additional guidelines and more detail about how to request an IETF non-working group email discussion list.
